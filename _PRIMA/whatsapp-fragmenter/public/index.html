Ayuda

¿Cómo usar?

  1. Exporta tu chat de WhatsApp como archivo .txt
  2. Sube el archivo aquí
  3. Configura las opciones (opcional)
  4. Espera a que se procese
  5. Descarga los fragmentos mensuales

¿Cómo exportar desde WhatsApp?

Android: Chat → Más opciones → Más → Exportar chat

iPhone: Chat → Editar → Más opciones → Exportar chat

¿Cómo usar en NotebookLM?

  1. Abre NotebookLM
  2. Crea un nuevo notebook
  3. Para cada archivo:
    • Click en "Add Source"
    • Sube el archivo Markdown
  4. ¡Listo! Puedes hacer preguntas sobre todo el chat

¿Qué se filtra?

  • Mensajes de sistema (si está activado)
  • Multimedia: [Imagen], [Video], [Audio], etc. (si está activado)
  • Mensajes eliminados

Limitaciones

  • Máximo 2GB por archivo
  • Solo formato .txt
  • Se procesan solo mensajes de texto

Acerca de

WhatsApp Chat Fragmenter v1.0.0

Herramienta para fragmentar archivos grandes de chat de WhatsApp en trozos mensuales.

Características

  • ✅ Detección automática de formato (ES, EN, PT, FR)
  • ✅ Fragmentación por mes
  • ✅ Filtrado de ruido
  • ✅ Índice automático
  • ✅ Procesamiento 100% local

Uso

Herramienta web con Node.js y Express.js

Privacidad

Todos los datos se procesan localmente en tu navegador y servidor. No se envía nada a internet. Tus datos son completamente privados.
